返回值内容

之前我们获取的rResponse对象中，具体包含什么内容呢。

text

服务器内容，大多数Unicode字符集都被无缝解码。

encoding

在发送请求时，会对返回值text进行编码猜测，猜测编码类型。使用encoding可以查看和修改编码类型

content

将返回正文作为字节，而非文本请求

json()

如果JSON解码异常会报ValueError错误。有时调用JSON成功并不能证明请求成功，需要使用status_code进行验证。

raw

如果希望从服务器获取原始socket响应可以使用raw

status_code/raise_for_status

响应状态码

headers

获取请求头信息

自定义请求头

如果需要向请求添加请求头，只需向headers参数传递一个字典就可以。
