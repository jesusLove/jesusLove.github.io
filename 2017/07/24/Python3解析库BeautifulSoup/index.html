Python3и§Јжһҗеә“BeautifulSoup

еҸ‘иЎЁдәҺ 2017-07-24 | еҲҶзұ»дәҺ Python3 | йҳ…иҜ»ж¬Ўж•°:

дёҠдёҖзҜҮпјҢеӯҰд№ еҰӮдҪ•иҜ·жұӮзҪ‘з»ңж•°жҚ®пјҢжҺҘдёӢжқҘеӯҰд№ дҪҝз”ЁBeautiful Soupи§ЈжһҗиҜ·жұӮеҲ°зҡ„ж•°жҚ®гҖӮ
Beautiful SoupжҳҜдёҖдёӘеҸҜд»Ҙд»ҺHTMLе’ҢXMLж–Үд»¶дёӯжҸҗеҸ–ж•°жҚ®зҡ„Pythonеә“гҖӮ

зүҲжң¬пјҡ4.4.0

е®үиЈ…Beautiful Soup

зЎ®дҝқе®үиЈ…Python3д№ӢеҗҺпјҢеҸӘйңҖдёҖиЎҢе‘Ҫд»ӨгҖӮ

жіЁж„ҸпјҢMacдёӯеҸҜиғҪйңҖиҰҒдҪҝз”Ёpip3 install beautifulsoup4

е®үиЈ…е®ҢBeautifulSoupеҗҺпјҢжҲ‘д»¬иҝҳйңҖиҰҒHTTPи§ЈжһҗеҷЁпјҢдҫӢеҰӮдёүж–№и§ЈжһҗеҷЁlxml

дёҮдәӢдҝұеӨҮеҸӘж¬ дёңйЈҺпјҒ

еҝ«йҖҹејҖе§Ӣ

еҜ№иұЎз§Қзұ»

BeautifulSoupе°ҶеӨҚжқӮзҡ„HTMLж–ҮжЎЈиҪ¬дёәдёҖдёӘеӨҚжқӮзҡ„ж ‘еҪўз»“жһ„пјҢжҜҸдёӘиҠӮзӮ№йғҪжҳҜPythonеҜ№иұЎпјҢжүҖжңүеҜ№иұЎеҸҜд»ҘеҪ’зәідёәеӣӣз§ҚпјҡTag, NavigableString, BeautifulSoup, CommentгҖӮ

Tag

TagеҜ№иұЎдёҺXMLе’ҢHTMLеҺҹз”ҹж–ҮжЎЈдёӯзҡ„tagзӣёеҗҢгҖӮдҫӢеҰӮпјҡ

дёӢйқўд»Ӣз»ҚдёӨдёӘжңҖйҮҚиҰҒзҡ„еұһжҖ§пјҡname е’Ң attributes гҖӮTagжңүеҫҲеӨҡеұһжҖ§е’Ңж–№жі•пјҢеңЁйҒҚеҺҶж–ҮжЎЈж ‘е’Ңжҗңзҙўж–ҮжЎЈж ‘дёӯиҜҰз»Ҷд»Ӣз»ҚгҖӮ

Name

дҪҝз”Ё.nameиҺ·еҸ–е’Ңдҝ®ж”№tagзҡ„еҗҚеӯ—

Attributes

дёҖдёӘtagжңүеҫҲеӨҡеұһжҖ§гҖӮдҫӢеҰӮпјҡеүҚйқўзҡ„tag<b class="boldest">,жңүдёҖдёӘclassеұһжҖ§гҖӮ

иҺ·еҸ–жүҖжңүзҡ„еұһжҖ§

еҸҰеӨ–tagзҡ„еұһжҖ§еҸҜд»Ҙж·»еҠ пјҢеҲ йҷӨе’Ңдҝ®ж”№гҖӮж“ҚдҪңж–№жі•е’Ңеӯ—е…ёдёҖж ·

жіЁж„ҸпјҡеӨҡеҖјеұһжҖ§пјҢдёҖдёӘеұһжҖ§еҸҜд»ҘеҗҢж—¶еӯҳеңЁеӨҡдёӘеҖј

NavigableString

еӯ—з¬ҰдёІеёёиў«еҢ…еҗ«еңЁtagдёӯпјҢдҪҝз”ЁNavigableStringзұ»жқҘеҢ…иЈ…tagдёӯзҡ„еӯ—з¬ҰдёІпјҡ

BeautifulSoup

BeautifulSoupеҜ№иұЎе№¶дёҚжҳҜзңҹжӯЈзҡ„HTMLжҲ–XMLзҡ„tagпјҢжүҖд»Ҙе®ғжІЎжңүnameе’ҢAttributeеұһжҖ§гҖӮжңүж—¶жҲ‘д»¬йңҖиҰҒ.nameжҹҘзңӢпјҢжүҖд»Ҙе®ғеҢ…еҗ«дёҖдёӘеҖјдёә[documnet]зҡ„зү№ж®ҠеұһжҖ§.name

Comment

дёҠйқўдёүдёӘиҰҶзӣ–дәҶHTMLе’ҢXMLдёӯзҡ„жүҖжңүеҶ…е®№гҖӮдҪҶжҳҜиҝҳжңүдёҖдәӣзү№ж®ҠеҜ№иұЎгҖӮ

Comment еҜ№иұЎжҳҜдёҖдёӘзү№ж®Ҡзұ»еһӢзҡ„NavigableStringеҜ№иұЎ

CommentеҜ№иұЎдјҡдҪҝз”Ёзү№ж®Ҡзҡ„ж јејҸиҫ“еҮәпјҡ

йҒҚеҺҶж–ҮжЎЈж ‘

рҹҢ°пјҡ

еӯҗиҠӮзӮ№

дёҖдёӘTagеҸҜиғҪеҢ…еҗ«еӨҡдёӘеӯ—з¬ҰдёІжҲ–е…¶е®ғзҡ„TagпјҢиҝҷдәӣйғҪжҳҜиҝҷдёӘTagзҡ„еӯҗиҠӮзӮ№гҖӮBeautiful SoupжҸҗдҫӣдәҶи®ёеӨҡж“ҚдҪңе’ҢйҒҚеҺҶеӯҗиҠӮзӮ№зҡ„еұһжҖ§гҖӮ

Tagзҡ„еҗҚеӯ—

ж“ҚдҪңж–ҮжЎЈж ‘жңҖз®ҖеҚ•зҡ„ж–№жі•е°ұжҳҜе‘ҠиҜүжғіиҺ·еҸ–ж Үзӯҫзҡ„еҗҚз§°пјҡ

.contents е’Ң .children

.contentsеұһжҖ§еҸҜд»Ҙе°Ҷtagзҡ„еӯҗиҠӮзӮ№д»ҘеҲ—иЎЁзҡ„ж–№ејҸиҫ“еҮәпјҡ

жіЁж„Ҹпјҡеӯ—з¬ҰдёІжІЎжңүеӯҗиҠӮзӮ№пјҢжүҖд»Ҙеӯ—з¬ҰдёІжІЎжңү.contentsеұһжҖ§гҖӮ

.descendants

.contentsе’Ң.childrenеұһжҖ§д»…еҢ…еҗ«tagзҡ„зӣҙжҺҘеӯҗиҠӮзӮ№гҖӮ.descendantsеұһжҖ§еҸҜд»ҘеҜ№жүҖжңүtagзҡ„еӯҗеӯҷиҠӮзӮ№иҝӣйҖ’еҪ’еҫӘзҺҜ

.string

.strings е’Ң stripped_strings

  • еҰӮжһңtagдёӯеҢ…еҗ«еӨҡдёӘеӯ—з¬ҰдёІпјҢеҸҜд»ҘдҪҝз”Ё.stringsжқҘеҫӘзҺҜиҺ·еҸ–гҖӮ
  • иҫ“еҮәзҡ„еӯ—з¬ҰдёІдёӯеҸҜд»ҘеҢ…еҗ«дәҶеҫҲеӨҡз©әж јжҲ–з©әиЎҢпјҢдҪҝз”Ё.stripped_stringsеҸҜд»ҘеҺ»йҷӨеӨҡдҪҷз©әзҷҪеҶ…е®№гҖӮ

зҲ¶иҠӮзӮ№

.parent

.parentеұһжҖ§иҺ·еҸ–жҹҗдёӘе…ғзҙ зҡ„зҲ¶иҠӮзӮ№гҖӮ

еӯ—з¬ҰдёІд№ҹжңүзҲ¶иҠӮзӮ№

зҡ„зҲ¶иҠӮзӮ№жҳҜBeautifulSoupеҜ№иұЎ
BeautifulSoupеҜ№иұЎзҡ„зҲ¶иҠӮзӮ№жҳҜNone

.parents

.parentsйҖ’еҪ’иҺ·еҸ–жүҖжңүзҡ„зҲ¶иҫҲиҠӮзӮ№гҖӮ

е…„ејҹиҠӮзӮ№

дҪҝз”Ё.next_siblingе’Ң.previous_siblingеұһжҖ§жқҘжҹҘиҜўе…„ејҹиҠӮзӮ№

йҖҡиҝҮ.next_siblingsе’Ң.previous_siblingsеұһжҖ§еҸҜд»ҘеҜ№еҪ“еүҚиҠӮзӮ№зҡ„е…„ејҹиҠӮзӮ№иҝӯд»Јиҫ“еҮәгҖӮ

жҗңзҙўж–ҮжЎЈж ‘

Beautiful Soupе®ҡд№үдәҶеҫҲеӨҡжҗңзҙўж–№жі•гҖӮдҫӢеҰӮпјҡfind() е’Ң find_all()гҖӮ

иҝҮж»ӨеҷЁ

еёёи§Ғзҡ„иҝҮж»ӨеҷЁзұ»еһӢпјҢеҰӮдёӢеҮ з§Қпјҡ

еӯ—з¬ҰдёІ

жңҖз®ҖеҚ•зҡ„иҝҮж»ӨеҷЁпјҢдҫӢеҰӮпјҡжҹҘжүҫ<b>ж ҮзӯҫеҸҜд»ҘеҶҷжҲҗfind_all('b')гҖӮ

жӯЈеҲҷиЎЁиҫҫејҸ

еҢ№й…Қз¬ҰеҗҲжӯЈеҲҷиЎЁиҫҫејҸзҡ„еҶ…е®№гҖӮ

еҲ—иЎЁ

еҢ№й…ҚеҲ—иЎЁдёӯжүҖжңүе…ғзҙ еҶ…е®№гҖӮ

TRUE

еҸҜд»ҘеҢ№й…Қд»»дҪ•еҖјгҖӮ

ж–№жі•

еҸҜд»Ҙе®ҡд№үдёҖдёӘжҺҘеҸ—дёҖдёӘеҸӮж•°зҡ„ж–№жі•пјҢиҝ”еӣһеёғе°”зұ»еһӢгҖӮеҰӮжһңжҳҜTRUEиЎЁзӨәеҪ“еүҚе…ғзҙ еҢ№й…ҚжүҫеҲ°пјҢеҗҰеҲҷдёәжүҫеҲ°гҖӮ

find_all

`find_all( name , attrs , recursive , string , kwargs )`**

жҗңзҙўжүҖжңүеҪ“еүҚtagзҡ„жүҖжңүtagеӯҗиҠӮзӮ№пјҢе№¶еҲӨж–ӯжҳҜеҗҰз¬ҰеҗҲиҝҮж»ӨеҷЁзҡ„жқЎд»¶гҖӮ

  1. name

nameеҸӮж•°еҸҜд»ҘжҹҘжүҫжүҖжңүеҗҚеӯ—дёәnameзҡ„tagгҖӮ

  1. keywordеҸӮж•°

еҰӮжһңдёҖдёӘжҢҮе®ҡеҗҚеӯ—зҡ„еҸӮж•°дёҚжҳҜжҗңзҙўеҶ…зҪ®зҡ„еҸӮж•°еҗҚпјҢжҗңзҙўж—¶дјҡжҠҠиҜҘеҸӮж•°еҪ“дҪңжҢҮе®ҡеҗҚеӯ—tagзҡ„еұһжҖ§жқҘжҗңзҙўгҖӮ

  1. жҢүClassжҗңзҙў

жҢүз…§зұ»еҗҚжҗңзҙўпјҢдҪҶжҳҜз”ұдәҺclassжҳҜдҝқз•ҷеӯ—пјҢжүҖд»ҘдҪҝз”Ёclass_д»ЈжӣҝгҖӮ

  1. StringеҸӮж•°

дҪҝз”ЁstringеҸӮж•°жҗңзҙўе’ҢдҪҝз”ЁnameеҸӮж•°зҡ„еҸҜйҖүеҖјдёҖж ·гҖӮ

  1. limitеҸӮж•°

дҪҝз”ЁlimitйҷҗеҲ¶иҝ”еӣһзҡ„ж•°йҮҸ

  1. recursiveеҸӮж•°

е°Ҷrecursiveи®ҫзҪ®дёәFalse,еҸӘдјҡжҗңзҙўtagзҡ„зӣҙжҺҘеӯҗиҠӮзӮ№гҖӮ

find

find()е’Ңfind_all()дёҚеҗҢзҡ„жҳҜпјҢеүҚиҖ…зӣҙжҺҘиҝ”еӣһз»“жһңпјҢеҗҺиҖ…иҝ”еӣһеҢ…еҗ«еҖјзҡ„еҲ—иЎЁгҖӮ

CSSйҖүжӢ©еҷЁ

еңЁBeautifulSoupеҜ№иұЎзҡ„select()ж–№жі•дј е…Ҙеӯ—з¬ҰдёІеҸӮж•°пјҢеҚіеҸҜд»ҘдҪҝз”ЁCSSйҖүжӢ©еҷЁгҖӮ
